jade中的注意点

    缩进代表 元素 的 包含关系，由于有些时候想输出出现了一些特殊符号的语义，
    
    
    内容有多个 标志:

    | :表示后面的内容 原样输出 ，不进行 编译成 元素
    . :表示 在此包含里的 所有 内容 都原样输出
    #{} :表示 大括号内 是 变量 ，可以 做一些表达式
    - :在 jade 文件中 定义 变量
    = :表示可以给 标签的 内容直接 加 内容 span=a
    include :表示 可以 像 引用模块一样引用文件 如：include ./includes/head.jade

    循环：见jade
